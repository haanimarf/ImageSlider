🖼️ Image Slider

A simple and responsive Image Slider (Carousel) built using HTML, CSS, and JavaScript.
This project allows users to navigate through multiple images using next and previous controls.

🚀 Features

⬅️ Next and Previous buttons

🔄 Smooth image transitions

📱 Responsive design

⚡ Lightweight and fast

🎯 Easy to customize


🛠️ Technologies Used

HTML5

CSS3

JavaScript

🎯 How It Works

Images are stored inside the images folder.

JavaScript controls image navigation.

CSS handles layout and animation effects.
